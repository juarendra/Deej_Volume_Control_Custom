# Deej_Volume_Control_Custom
Documentation Product Volume Control Using Software Deej

## Spesification
- Microcontroller STM32F103
- Bootloader STM32duino
- 5 Potensio 
- 5 Button 
- Case 3D Print
- USB type C
- Compatible with Deej
